Workflow

Name Description
Workflow type STANDARD – single approval.
States
  • Initial: content is just created and not persisted yet.
  • Waiting: content has been submitted and waits for approval.
  • Approved: content has been approved and is available online.
  • Suspended: content has been approved, but is temporarily offline.
  • Deleted: content has been deleted from the database.
Users are able to manage and edit their own data
If creator is deleted the "admin" user will be assigned.
If last editor is deleted the "admin" user will be assigned.
